# system-core — MASTER BRAIN

זהו ריפו-העל (Single Source of Truth) של כל המערכת של יניב.

## תפקידים
- **system-core**: מקור אמת, אסטרטגיה, חוקים, דשבורדים, מצב (STATE)
- **my-assistant**: מנוע ביצוע (Execution Engine) שמושך חוקים ומיישם פעולות

## מה מושכים למנוע
קבצי ה-Master Pack נמצאים בשורש הריפו וב-STATE/.
המנוע מושך אותם לתיקייה _master/ בתוך my-assistant.
